# React Project

This is a full-stack React starter app, meant for practicing and prototyping.

## ✨ Requirements

1. Node >= 18.x
2. Git Bash (Windows users)

## ✨ Usage

1. `npm install`
2. `npm run dev`
3. `npm test`

See the [package.json file](./package.json) for other scripts.
